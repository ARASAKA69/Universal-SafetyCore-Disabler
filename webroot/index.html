About The Module

            
            
What is SafetyCore?

            
SafetyCore is part of Google's 'Private Compute Core', a system for on-device AI features like Live Captions, Now Playing, and scanning images for sensitive content warnings.

            
            
How does this module work?

            
This module intelligently detects if SafetyCore is a System App or a User App on your device. It then replaces the original application with a completely empty, non-functional placeholder.

            
            
Why is this safe from updates?

            
The placeholder app has a different cryptographic signature than Google's official app. This means the Play Store cannot "update" it, guaranteeing that future versions of SafetyCore won't be pushed to your device while this module is active.
